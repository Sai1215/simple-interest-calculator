# Simple Interest Calculator

This is a simple project to calculate Simple Interest.

## Formula
The formula used for calculating simple interest is:
$$\text{Simple Interest} = \frac{P \times R \times T}{100}$$

Where:
* **P** = Principal amount
* **R** = Annual interest rate
* **T** = Time (in years)

## Input
The user typically inputs the Principal, Rate, and Time.

## Output
The program outputs the calculated simple interest.
